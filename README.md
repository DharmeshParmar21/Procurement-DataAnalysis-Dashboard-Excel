## ğŸ“˜ Project Overview  
Procurement data refers to the information and records related to the process of acquiring goods and services for an organization.  
It is crucial for managing purchases, evaluating vendors, tracking expenses, and ensuring efficient supply chain operations.

This project uses **Excel** and **Power Query** to analyze purchase orders, vendor performance, inventory movements, and procurement trends.  
The dashboard provides a complete visualization of procurement activity for better business decision-making.
